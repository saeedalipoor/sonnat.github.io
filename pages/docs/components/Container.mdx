</h2>

<p>
  Fixed container is a responsive, fixed-width container, meaning its{" "}
  <inlineCode>max-width</inlineCode> changes at each breakpoint.
</p>

```tsx
<Container>{/* Your content */}</Container>
```

<h2>
  <span>Fluid Container</span>
  <AnchorButton anchorId="fluid-container" />
</h2>

<p>
  Fluid container is a full width container, spanning the entire width of the
  viewport.
</p>

```tsx
<Container fluid>{/* Your content */}</Container>
```
