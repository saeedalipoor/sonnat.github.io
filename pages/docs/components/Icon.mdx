<p>
  You can use this component with any other icon package that exports the svg
  paths.
  <br />
  <inlineCode>@sonnat/icons</inlineCode> exports both the paths (access via{" "}
  <inlineCode>@sonnat/icons/paths</inlineCode>) and also the svg icons. These
  svg icons exported from <inlineCode>@sonnat/icons</inlineCode> use the{" "}
  <inlineCode>@sonnat/ui/Icon</inlineCode> component to render the SVG path for
  each icon, and so they have a peer-dependency on the next release of the{" "}
  <inlineCode>@sonnat/ui</inlineCode>.
</p>
