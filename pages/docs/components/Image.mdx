<h2>
  <span>Layout</span>
  <AnchorButton anchorId="layout" />
</h2>

<p>
  The layout behavior of the image as the viewport changes size. Defaults to{" "}
  <inlineCode>intrinsic</inlineCode>.
</p>
<p>
  When <inlineCode>fixed</inlineCode>, the image dimensions will not change as
  the viewport changes (no responsiveness) similar to the native{" "}
  <inlineCode>img</inlineCode> element.
</p>
<p>
  When <inlineCode>intrinsic</inlineCode>, the image will scale the dimensions
  down for smaller viewports but maintain the original dimensions for larger
  viewports.
</p>
<p>
  When <inlineCode>responsive</inlineCode>, the image will scale the dimensions
  down for smaller viewports and scale up for larger viewports.
</p>
<p>
  When <inlineCode>fill</inlineCode>, the image will stretch both width and
  height to the dimensions of the parent element, provided the parent element is
  relative. This is usually paired with the <inlineCode>object-fit</inlineCode>{" "}
  css property.
</p>
